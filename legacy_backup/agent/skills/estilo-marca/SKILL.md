# Skill: Estilo y Marca
Marca: Skiveld (Software de automatizaciones y asesoría en marketing)

## Cuándo usar este Skill
- Si vas a diseñar una UI o una pantalla para la plataforma de software.
- Si vas a crear una landing page de servicios o producto.
- Si vas a escribir textos: titulares, CTAs, botones, mensajes de error, emails o descripciones.
- Si vas a generar assets visuales o código “de cara al usuario final”.

## Regla número 1
No improvises el estilo ni los colores. La marca Skiveld tiene una paleta definida basada en su logotipo (Azul, Negro, Verde para destaques, acentos en Rojo). Si falta algún dato de diseño o de texto, usa los valores definidos en los recursos adjuntos.

## Dónde mirar según el tipo de tarea
- Estilo visual (colores extraídos del logo, tipografías, espaciado): recursos/estilo-visual.json
- Forma de escribir (tono de agencia tecnológica, estructura y vocabularios útiles): recursos/guia-de-textos.md
- Decisiones técnicas (framework, estilos, librerías correspondientes): recursos/reglas-tecnicas.md

## Checklist antes de entregar
1) ¿Se siente como un producto/web de Skiveld (tecnológico, profesional, limpio)?
2) ¿Los colores respetan la paleta extraída del logo (Azul corporativo principal, Verde para éxito o destacar, Negro para alto contraste)?
3) ¿El tono escrito proyecta autoridad en automatización y marketing sin perder de vista los beneficios para el cliente?
4) ¿La jerarquía visual guía claramente al usuario hacia la conversión o la siguiente acción?

## Cómo mejorar este Skill
Si notas que Antigravity genera diseños o textos que no cuadran con tu visión de Skiveld, no lo corrijas solo en el prompt ocasional: entra a los archivos de la carpeta `recursos/`, ajusta las guías y vuelve a generar para que la "fuente de verdad" quede siempre actualizada conforme evoluciona tu empresa.
